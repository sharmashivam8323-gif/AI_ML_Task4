# 🤖 AI & Machine Learning – Task 4

## Classification Models, Evaluation Metrics & Handling Imbalanced Data

### 📌 Objective

Build a binary classification model using the Breast Cancer Wisconsin Dataset and evaluate it using multiple classification metrics.

---

## 📊 Dataset

Breast Cancer Wisconsin Dataset

Target Classes:
- 0 → Malignant
- 1 → Benign
